BCA-401 Advanced Java
Unit 1: Swing & AWT

Introduction to AWT, Event Handling, Swing Components, Layout Managers, Graphics Programming.

Unit 2: JDBC

Database Connectivity, JDBC Drivers, Connection Interface, Statement & PreparedStatement, ResultSet.

Unit 3: Servlets

Servlet Life Cycle, Request & Response, Session Tracking, Cookies, URL Rewriting.

References

1. "Java: The Complete Reference" by Herbert Schildt

2. "Head First Servlets and JSP" by Bryan Basham

BCA-402 Web Technology
Unit 1: HTML5 & CSS3

Semantic Elements, Forms, Canvas, SVG, CSS Selectors, Flexbox, Grid Layout, Responsive Design.

Unit 2: JavaScript

DOM Manipulation, Event Listeners, ES6 Features (Arrow functions, Promises), AJAX, Fetch API.

Unit 3: Server Side (PHP)

PHP Syntax, Form Handling, Database Connection (MySQL), Sessions & Cookies.

References

1. "HTML & CSS: Design and Build Websites" by Jon Duckett

2. "Learning PHP, MySQL & JavaScript" by Robin Nixon

BCA-403 Software Engineering
Unit 1: Process Models

SDLC, Waterfall Model, Spiral Model, Agile Methodology, Scrum, Extreme Programming.

Unit 2: Requirement Analysis

SRS Document, Functional & Non-functional Requirements, Use Case Diagrams.

Unit 3: Testing

Black Box vs White Box Testing, Unit Testing, Integration Testing, System Testing.

References

1. "Software Engineering: A Practitioner's Approach" by Pressman

2. "Software Engineering" by Ian Sommerville

BCA-404 Data Communications
Unit 1: Fundamentals

OSI Model, TCP/IP Model, Transmission Media (Guided/Unguided), Switching Techniques.

Unit 2: Data Link Layer

Framing, Error Control, Flow Control, Ethernet, CSMA/CD, MAC Addresses.

Unit 3: Network Layer

IPv4/IPv6 Addressing, Subnetting, Routing Algorithms (Distance Vector, Link State).

References

1. "Data Communications and Networking" by Behrouz A. Forouzan

2. "Computer Networks" by Andrew S. Tanenbaum

BCA-405 Database Mgmt Systems
Unit 1: Relational Model

E-R Diagrams, Relational Algebra, Tuple Calculus, Keys (Primary, Foreign, Candidate).

Unit 2: Normalization

Functional Dependency, 1NF, 2NF, 3NF, BCNF, De-normalization.

Unit 3: SQL & Transactions

DDL, DML, DCL, Joins, Views, ACID Properties, Concurrency Control.

References

1. "Database System Concepts" by Korth and Silberschatz

2. "Fundamentals of Database Systems" by Elmasri & Navathe

BCA-406 Operating Systems
Unit 1: Process Mgmt

Process Scheduling, Scheduling Algorithms (FCFS, SJF, Round Robin), Threads.

Unit 2: Synchronization

Semaphores, Mutex, Deadlocks (Prevention, Avoidance, Detection).

Unit 3: Memory Mgmt

Paging, Segmentation, Virtual Memory, Page Replacement Algorithms.

References

1. "Operating System Concepts" by Galvin and Silberschatz

2. "Modern Operating Systems" by Andrew S. Tanenbaum

BCA-407 Computer Graphics
Unit 1: Output Primitives

Line Drawing Algorithms (DDA, Bresenham), Circle Drawing, Scan-line Polygon filling.

Unit 2: Transformations

2D Transformations (Translation, Rotation, Scaling), Homogeneous Coordinates, Clipping.

Unit 3: 3D Concepts

3D Object Representation, Projections (Parallel, Perspective), Hidden Surface Removal.

References

1. "Computer Graphics C Version" by Donald Hearn & M. Pauline Baker

2. "Interactive Computer Graphics" by Edward Angel

BCA-408 Numerical Methods
Unit 1: Roots of Equations

Bisection Method, Regula-Falsi Method, Newton-Raphson Method.

Unit 2: Linear Equations

Gauss Elimination, Gauss-Jordan Method, Gauss-Seidel Iteration.

Unit 3: Integration

Trapezoidal Rule, Simpson's 1/3 and 3/8 Rules, Runge-Kutta Methods.

References

1. "Introductory Methods of Numerical Analysis" by S.S. Sastry

2. "Numerical Methods for Engineers" by Steven C. Chapra
