Full Day Activity
BCA23205T Artificial Intelligence
Module 1

Introduction to AI, Scope, History, Intelligent Agent & Environment, Structure of Agent, Problem solving Agent.

Module 2

Problem Solving: Formulating Problem, 8 Puzzle, Missionaries and Cannibal, Searching (Informed/Uninformed).

Module 3

Knowledge Representation: Propositional Logic, First Order Logic, Syntax, Semantics, Inference Rules.

Module 4

Expert Systems: Definition, Characteristics, Components, and Applications.

Text Book

"Artificial Intelligence: A Modern Approach" by Russell & Norvig

BCA23206T Computer Architecture
Unit 1

Computer Org: Instruction codes, registers, bus, instruction cycle, control.

Unit 2

CPU Org: Stack organization, RISC/CISC, Addressing modes, Pipelining, Flynn's classification.

Unit 3

I/O Org: Peripheral devices, Interfaces, Asynchronous data transfer, DMA, Priority interrupt.

Unit 4

Memory Org: Hierarchy, Associative memory, Cache memory, Virtual memory.

Unit 5

Microprocessor: 8085 Architecture, Pin diagram, Addressing modes, Microcontrollers.

Text Book

"Computer System Architecture" by M. Morris Mano

BCA23207T Advanced Web Tech
Module I

Intro to JS: Client/Server side, Lexical structure, Variables, Operators, Control flow.

Module II

Objects & Inheritance: Creating objects, Accessors, Prototypes, Classes, Constructors.

Module III

Arrays: Declaring, Accessing, Built-in methods, Iterators, Nesting.

Module IV

DOM: BOM vs DOM, DOM Tree, Node properties, Event Handling.

Module V

Built-in Objects: String, Math, Date, RegExp, Maps, Sets, JSON, Error Handling.

Text Book

"JAVASCRIPT Syntax and Practices" by Dangi S.

BCA23208T Machine Learning
Module I

Intro: Types of ML (Supervised, Unsupervised, Reinforcement), Data Preprocessing, EDA.

Module II

Supervised Learning: Regression, KNN, SVM, Nave Bayes, Decision Tree.

Module III

Unsupervised Learning: K-means, Hierarchical Clustering, PCA.

Module IV

Model Performance: Evaluation metrics (Accuracy, Precision, Recall), Cross-validation.

Text Book

"Machine Learning" by Tom M. Mitchell

BCA23209T Mathematics
Module I

Sets, Relations & Functions: Venn diagrams, Equivalence relations, One-to-one/Onto functions.

Module II

Determinants & Matrices: Rank, Eigenvalues, Eigenvectors, Cayley-Hamilton theorem.

Module III

Combinatorics: Permutations, Combinations, Recurrence relations.

Module IV

Sequence & Series: AP, GP, Convergence, Divergence, Infinite series.

BCA23210T Mobile App Dev
Module I

Intro to Android: SDK, Android Studio, Manifest file, App Anatomy.

Module II

App Setup: Activities, Services, Intents, Permissions.

Module III

UI Design: Layouts, Screen elements, Animation.

Module V

Storage & APIs: SQLite, Content Providers, Networking APIs.

Text Book

"Android Wireless Application Development" by Lauren Darcey
